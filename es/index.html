Sobre mi /

¡Hola! Me llamo Roger y me apasiona el diseño y el desarrollo web. Me considero una persona responsable, generosa y dispuesta a ayudar siempre que sea necesario. Uno de mis puntos fuertes es la resiliencia delante de determinadas situaciones.

En cuanto a mi experiencia laboral, durante dos años he estado trabajando con CMS como WordPress y PrestaShop, así como modificando su diseño con CSS. Actualmente, estoy estudiando el Grado Universitario de Técnicas de Interacción Digital y Multimedia a través de la UOC para especializarme profesionalmente. Mi objetivo es aprender constantemente para mejorar mis habilidades.

¡Gracias por tu tiempo!

Proyectos recientes /

Meritxell Egea

https://www.meritxellegea.com/

Proyecto personal para Meritxell Egea Fisioterapia y espacio Karuna en Malgrat de Mar. Sitio hecho con WordPress y la plantilla OceanWP con modificaciones en el CSS y diferentes plugins. Las fotos del espacio Karuna también son realizadas por mí.

Desk and Sit

https://deskandsit.com/

PrestaShop realizado trabajando en Seoagil. El proyecto consistió en pasar un PrestaShop 1.6 a la nueva versión 1.7. Se migraron los productos, pedidos, clientes y módulos. Se realizaron modificaciones de ficheros .tpl y el CSS de la plantilla Warehouse.

Flexxit

https://flexxit.es

PrestaShop realizado trabajando en Seoagil. Web creada desde cero en PrestaShop 1.7 y plantilla Warehouse. Se importaron productos y se modificó el CSS para adaptar el diseño al que había enviado el cliente en un fichero PDF.

Gelats i torrons xixona

https://gelatsitorronsxixona.com/

WordPress realizado trabajando en Seoagil. Proyecto en WordPress y utilizando la plantilla Divi y WooCommerce. Se hicieron modificaciones en el CSS según las propuestas del cliente. Contiene plugins para modificar las tarifas de envío.

Fotomatón Low Cost

https://fotomatonlowcost.com/

WordPress realizado trabajando en Seoagil. El proyecto consistió en modificar el diseño de un WordPress ya existente. Se modificó el CSS para que la plantilla se adaptara al diseño propuesto por el cliente que envió en un PDF.

Portfolio personal

Portfolio personal como proyecto propio realizado con HTML, CSS y Javascript. Tanto el diseño responsive como la maquetación es propia i no se han utilizado plantillas. El código puede verse el mi repositorio personal de GitHub.
